Setup Email Backup (Optional)

                
If you lose access to your authenticator, you can receive backup codes via email.


                

                    
EmailJS Setup Instructions:

                    
    
                        
  1. Go to EmailJS.com and create a free account
    2. 
                        
  2. Add an email service (Gmail, Outlook, etc.)
    3. 
                        
  3. Create an email template with these variables:
                            
      
                                
    • {{to_email}} - Recipient email
      • 
                                
    • {{backup_code}} - The 6-digit code
      • 
                                
    • {{expiry_minutes}} - Code validity time
      • 
                                
    • {{timestamp}} - When code was sent
      • 
                            
    
                        
    4. 
                        
  4. Copy your Public Key, Service ID, and Template ID
    5. 
                    


                    

                        
Sample Email Template:

                        
Subject: Admin Login Backup Code

Your backup code is: {{backup_code}}

This code is valid for {{expiry_minutes}} minutes.
Requested at: {{timestamp}}

If you didn't request this code, please ignore this email.

                    


                    

                        
Your EmailJS Configuration:

                        
After setting up EmailJS, add these values to your twofa.js file:

                        

                            
EMAILJS_PUBLIC_KEY: 'your-public-key',
EMAILJS_SERVICE_ID: 'your-service-id',
EMAILJS_TEMPLATE_ID: 'your-template-id',
BACKUP_EMAIL: 'your@email.com',
